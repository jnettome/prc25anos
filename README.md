# prc25anos

Projeto criado como prova de conceito para o sistema de venda de vouchers e controle de uso dos mesmos para o aniversário de 25 anos do Paraná Clube, movido pela torcida Fúria Independente.

Processamento de pagamentos via iugu.com. Emails disparados por Mandrill.

## dependências

    ruby 2.1.1

### contribuindo

    git clone git@github.com:jnettome/prc25anos.git
    cd prc25anos
    bundle install
    rake db:setup
    rails server

### testes

    rake
